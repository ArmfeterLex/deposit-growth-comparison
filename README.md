# Deposit Growth Comparison

Консольное приложение на C# для сравнения роста вклада по двум банковским схемам.

## Функциональность

- Ввод начальной суммы вклада
- Ввод процентной ставки
- Расчет по двум схемам:
  - Банк 1 (простые проценты) — на начальную сумму
  - Банк 2 (сложные проценты) — на текущую сумму
- Вывод по годам до удвоения вклада

## Пример работы

Введите начальный вклад (от 100 до 500 руб): 100
Введите процентную ставку (от 10 до 30): 20
год 1, банк1 = 120, банк2 = 120
год 2, банк1 = 140, банк2 = 144
год 3, банк1 = 160, банк2 = 172.8
год 4, банк1 = 180, банк2 = 207.36

## Куда можно встроить

- В обучающие материалы по процентам
